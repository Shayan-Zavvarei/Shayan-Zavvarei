## 🚀 What I'm Doing Now

- 🌌 **Master's Thesis @ [Computational Cosmology Group, SBU](https://github.com/Shayan-Zavvarei/CCG-SBU)** — Studying the impact of massive neutrinos on large-scale cosmic structure using the [Quijote simulation suite](https://github.com/Shayan-Zavvarei/Thesis-Quijote-Simulation), running N-body simulations across cosmological models and analyzing matter power spectrum suppression with simulation-based inference and topological data analysis.
- 🤖 **AI for Physics @ IPM** — Building [DingoT1-Lite](https://github.com/Shayan-Zavvarei/DingoT1-Lite), a lightweight implementation of DINGO-T1 for fast gravitational-wave parameter estimation using transformers + normalizing flows for neural posterior estimation on LIGO/Virgo data.
- 🌱 **Learning** — Numerical relativity, advanced particle-physics analysis, and high-performance computing for simulation workloads.
